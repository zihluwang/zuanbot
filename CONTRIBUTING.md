# Contributing

本项目基于 Tauri v2.0 开发，技术栈如下：

**后端**

- Rust 1.88.0

**前端**

- React v18
- Tailwind CSS v4

欢迎您从**代码**或**内容**两方面参与贡献，共同完善本项目。

## 贡献指引

### 代码贡献

1. 【可选】创建 Issue，讨论你的想法。
2. Fork 本仓库。
3. 基于 `develop` 分支创建新 `feature/<your_feature_name>` 分支。
4. 开始完善你的功能。
5. 测试功能。
6. 提交你的代码并通过创建 Pull Request 将代码合并到 `develop` 分支。
7. 等待上线。

### 内容贡献

**暂时还没想好怎么提交新内容，等待这里更新再说吧。**
